Provides basic compatibility with bbPress and the Newsfront theme with a few extra goodies.

== Description ==

bbPress Genesis Extend does some behind-the-scenes tweaks and fixes to make bbPress/NewsFront integration as quick and painless as possible.

Additionally, this plugin also:

* Adds option for a forum specific sidebar
* Adds option to control the layout of your forum, separate from other pages
* Adds option to disable bbPress Forum and Topic descriptions.

The forum sidebar and layout options are located on the Theme options page, look for 'bbPress'.

== Installation ==

1. Upload `newsfront-bbpress` to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure optional settings on the Genesis Settings page
